# ONGC Monitoring Dashboard

A modern real-time industrial monitoring dashboard built for ONGC operations. This project provides live visualisation of oil and gas production data, well monitoring, environmental metrics, and safety analytics through an interactive and responsive dashboard.

---

#  Live Demo

 [DashBoard](https://ongc-deshboard.vercel.app)

---

# Features

- Real-time oil and gas production monitoring
- Live well pressure analytics
- Environmental monitoring dashboard
- Safety metrics and active alerts tracking
- Interactive charts and graphs
- Responsive modern UI
- Auto-refreshing dashboard data
- Production statistics by location
- Clean industrial dashboard design
